# f5h

PC-9800シリーズなどで多くの人に愛用されたファイラー FILMTN(K.Ishida氏作、1996年)の思い出をUNIX TUIに再現しようとしたものです。

設定ファイルの詳細は英語版 [README.md](../README.md) を参照してください。

---

## 経緯

現代のUNIX TUIでファイルメンテを使えたらうれしいのでClaude Codeに実装してもらいました。
hjklキーバインドなど、操作や表示の一部はUNIXの文脈に揃えてあるので再現というより再解釈と考えていただければ幸いです。

## 必要環境

- Rust (edition 2024)
- Linux / macOS
- [Nerd Fonts](https://www.nerdfonts.com/) 対応ターミナル（gitブランチアイコンの表示に必要）

## インストール

```bash
git clone https://github.com/mskako/f5h
cd f5h
cargo build --release
cp target/release/f5h ~/.local/bin/
```

## キー操作

| キー | 動作 |
|---|---|
| `h` / `←` | 左の列に移動 |
| `j` / `↓` | 下に移動 |
| `k` / `↑` | 上に移動 |
| `l` / `→` | 右の列に移動 |
| `g` / `G` | 先頭 / 末尾へ |
| `PageUp` / `PageDown` | 前ページ / 次ページ |
| `Enter` | ディレクトリ移動 / ファイルを開く |
| `Backspace` | 親ディレクトリへ |
| `/` | ホームディレクトリへ |
| `Space` | タグ付け + 1つ下に移動 |
| `Home` | 全ファイルのタグをトグル |
| `q` | 終了 |
| `!` / `@` / `#` / `%` | 1 / 2 / 3 / 5 列表示 |
| `Tab` | ディレクトリツリーペイン（トグル） |
| `e` | エディターで開く |
| `c` | 複写（タグ付きファイル、またはカーソル位置のファイル） |
| `m` | 移動 |
| `d` | 削除 |
| `n` | 名前変更 |
| `a` | 権限変更（8進数 chmod） |
| `K` | ディレクトリ作成 |
| `J` | ディレクトリジャンプ *(未実装)* |

矢印キーは常に hjkl の別名です（設定での変更不可）。上記以外のキーはすべて `~/.config/f5h/config.toml` の `[keys]` セクションで変更できます。
`g` / `G` はそれぞれ `first_entry` / `last_entry` の既定値なので、これらも `[keys]` で変更できます。
ディレクトリを指す symlink には `Enter` でそのまま入れます。ファイル情報欄には symlink のリンク先が表示されます。

### 複写・移動の競合ダイアログ

移動先に同名ファイルが存在する場合、FILMTN スタイルの競合ダイアログが表示されます。

| キー | 動作 |
|---|---|
| `u` | タイムスタンプが新しい場合のみ複写/移動 |
| `o` | 上書きで複写/移動 |
| `c` | 名前を変更して複写/移動 |
| `n` | スキップ |
| `U` `O` `N`（大文字） | 同上、ただし**以降すべての競合**に一括適用 |
| `Esc` | 操作を中断 |

## ファイルリストの見方

```
*M main.rs      49152  2026-03-15 12:00
^^
||
|+-- git ステータス: M=変更  A=追加  ?=未追跡  D=削除
+--- タグマーク: * = 選択中（黄色表示）
```

カーソル行はターミナルの反転表示で強調されます。タグマーク・gitステータスの3文字プレフィックス部分は反転しないので、選択状態が一目でわかります。

## 設定の概要

`~/.config/f5h/config.toml` に置きます。すべての項目は省略可能です。

```toml
[display]
show_hidden = false    # true にすると隠しファイル (.始まり) を表示

[colors]
border = "cyan"        # 枠の罫線
title  = "yellow"      # セクションタイトル
label  = "cyan"        # フィールドラベル（原作のインフォメーション・タイトルに相当）
unit   = "cyan"        # 単位 (Gi / Ki / %)
date   = "white"       # タイムスタンプ・時計
clock  = "white"       # 右上の時計
ls_colors = ""         # Linux では $LS_COLORS、macOS では $LSCOLORS を使用

[programs]
editor = "vim"         # $EDITOR → nano にフォールバック
pager  = "less"        # $PAGER  → less にフォールバック
# 拡張子ごとのプログラム指定（editor/pager と同列）
pdf    = "evince"
jpg    = "feh"
mp4    = "mpv"
```

キーバインドのカスタマイズや色の詳細な指定方法は [README.md](../README.md) を参照してください。
色分けは GNU/Linux では `LS_COLORS`、macOS では `LSCOLORS` を参照します。

## 今後の予定

- `J`: ディレクトリジャンプダイアログ
- ソートモード
- ワイルドカードフィルタ
- `F1`: ヘルプ画面
- 複写・移動ダイアログ: 入力欄空でEnterするとツリーピッカーが開く
